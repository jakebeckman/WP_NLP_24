MA498 Research Project

Abstract: Systematic reviews and meta-analyses in biomedicine represent a synthesis of literature in the current state of the field. To this end, review papers are typically registered in PROSPERO and require input of a primary outcome variable or variables. With the ability to scrape databases using code and then summarizing the results using large language models combined with machine learning methods, developing such a synthesis of the literature can look very different. Here, I will present a review of the literature on the association of genetics and physical activity. The PubMed database was scraped under key words associated with the topic. To be included, an article had to be associated with at least one key word from each of the following categories. The first was words associated with genetic makeup and genetic studies, including "genes", "genetics", "GWAS", "genomic", or "hereditary". The second was words associated with physical activity, including "exercise", "physical activity", "fitness", "training", or "aerobic". A result of 4,994 number of publications were obtained. The abstracts were then summarized and sorted into categories using natural language processing and machine learning algorithms and then visualized. I will report the process of searching, summarizing, grouping, and visualizing article collection using these methods as a more efficient and wholistic process of conducting a literature review.

Project 2: Topic Modeling

This project focuses on analyzing a dataset of over 400,000 Quora questions using topic modeling techniques. The objective is to identify 20 distinct categories for the unlabeled questions through preprocessing, vectorization, and clustering. The analysis uses tools such as TF-IDF, cosine similarity, and natural language processing methods for text lemmatization and cleaning.

Project 3: Classification

This project explores advanced natural language processing (NLP) techniques to analyze and classify text data. The CNN model classifies Amazon reviews by leveraging preprocessing steps like tokenization, lemmatization, and dataset balancing to improve accuracy. The Cosine Similarity model uses TF-IDF vectorization and clustering to identify patterns in text data, supplemented by visualizations like confusion matrices and word clusters. Together, these models demonstrate powerful methods for understanding and organizing text data effectively.
